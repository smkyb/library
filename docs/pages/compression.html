# compression 複数の`vector`を同時に座標圧縮します。元の`vector`の要素を0から始まる連続した整数に置き換えます。 ## compression ```cpp template<typename T, typename... Args> vector<T> compression(vector<T>& v, Args&... args) ``` 引数として与えられたすべての`vector`の要素を抽出し、それらを同時に座標圧縮します。各`vector`の各要素は、その値がソートされたユニークな集合内で何番目に出現するかを示す0から始まるインデックスに置き換えられます。元の`vector`は参照渡しのため、関数呼び出し後に内容が変更されます。
* **型パラメータ`T`**: `vector`の要素の型。比較可能である必要があります。 * `v`: 圧縮したい最初の`vector`です。 * `args`: 圧縮したい追加の`vector`です。複数指定可能です。
### 戻り値 `vector<T>`: 圧縮後の各値に対応する元の値のリストです。`C[i]`は圧縮後の値`i`に対応する元の値を示します。
### 制約 * `T`が比較可能であること (`operator<` が定義されていること)。 * `v`および`args`で与えられるすべての`vector`の要素の合計数が、システムメモリや許容される時間計算量の範囲内であること。 ### 計算量 全要素数を`N_total`としたとき、O(`N_total` log `N_total`)
